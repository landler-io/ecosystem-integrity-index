# Preprocessing Pipelines

Data preparation scripts for EII inputs.

## Subdirectories

- `ecoregions/` - Ecoregion rasterization
- `natural_lands/` - Identification of pristine reference areas
- `npp/` - NPP data processing and annual aggregation
- `protected_areas/` - WDPA rasterization
- `soilgrids/` - Soil property data preparation
- `utils/` - Shared preprocessing utilities
