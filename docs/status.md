# Project Status

EII is currently in **beta**. Core functionality is stable, but some datasets
and documentation are still evolving.

## Release checklist

- ✅ API surface for `eii.client` and `eii.compute` stabilized
- ✅ Core methodology documentation
- ⬜ Public data provenance and update cadence finalized
- ⬜ Final citation details (DOI / journal)
- ⬜ End-to-end reproducibility checklist published

## Known gaps

- Broader validation summaries are still being finalized.
- Training utilities are available but minimally documented.

If you need specific guarantees for a release, please open an issue describing
the required assurances.
